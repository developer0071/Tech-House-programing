ğŸª About Tech House

Tech House is a console-based Python application that simulates the operation of a household appliance store. The system supports two main user roles:

Customers, who can browse products, manage a shopping cart, and place orders.

Administrators, who can manage products, stock, customers, and sales data.

The application focuses on clarity, logic, and usability rather than graphical interfaces, making it ideal for learning backend logic and system structure.
